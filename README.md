# doc-agent

> Auto-generate and keep API docs in sync with your code.

## What it does

Reads source files, extracts public interfaces, and writes/updates Markdown or OpenAPI docs. Runs in CI — opens a PR when docs drift from code.

## Quickstart

```bash
pip install doc-agent-ai
```

## Usage

```bash
doc-agent generate src/ --output docs/ --format markdown
```
